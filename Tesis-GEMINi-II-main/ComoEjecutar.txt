Hay 3 modulos, disenador (para evitar problemas por tener "ñ" en el nombre de archivo), multiplexador y coordinador.
Los tres pueden ser ejecutados desde la linea de comandos. Se pueden mostrar los argumentos con "-h".
El unico argumento obligatorio son los nombres de archivo. Busca los archivos de secuencia en la carpeta "Genes" y los archivos de paneles de partidores en la misma carpeta donde se encuentra el módulo.
Para argumetnos que acepten listas, se puede puede repetir varias veces para agregar a la lista.
El archivo de fuente "OrkneyRegular.ttf" es utilizado por el módulo disenador.
Además se incluye un archivo ".yml" con el entorno de Conda utilizado en la ejecución.

Ejemplos de ejecución:


Coordinador para un gen:

    python coordinador.py -gfn IFT140.docx -tm 59 -marex 100 -lamp 480 -dgd -9000 -dgm -6000


Coordinador multigen:

    python coordinador.py -gfn COL4A3.docx -gfn COL4A4.docx -gfn COL4A5.docx -marex 150 -lamp 480 -mb 180 -dgd -9000 -dgm -6000


Multiplexador para un gen:

    python multiplexador.py -gfn IFT140.docx -pfn IFT140.csv -mdg -6000


Multiplexador multigen:

    python multiplexador.py -gfn COL4A3.docx -pfn f-1_COL4A3.csv -gfn COL4A4.docx -pfn f-1_COL4A4.csv -gfn COL4A5.docx -pfn f-1_COL4A5.csv -mdg -6000


Disenador:

    python disenador.py -gfn IFT140.docx -fotm 66 -mvc 50 -marex 100 -tmmi 59 -lamp 480 -dgd -9000
